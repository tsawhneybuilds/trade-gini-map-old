Country-size effect

Country Size Effect

This page reports the descriptive population-gradient test using the rd2 country sample. The main comparison is large versus small countries within the same year, controlling for log GDP per capita and year fixed effects.

Size effect

Country size and concentration

Question
Are larger countries less concentrated in product or partner trade baskets?
Supports yes if
The within-year cross-country coefficient on log population is consistently negative, especially for export product concentration.
Weakens yes if
The coefficient is unstable, flips sign, or only survives one standard-error convention.
Current answer
Mostly supports an export product-size gradient. The main, two-way-clustered, and Fama-MacBeth checks point to lower export product concentration among larger countries; import and partner gradients are weaker.

Reading the Test

  • The main estimand is descriptive: it compares larger and smaller countries within the same year, controlling for log GDP per capita and year fixed effects.
  • The strongest export gradient is Exports Product Top 1% share, with log-population coefficient -0.0379 and country-clustered SE 0.0088.
  • The strongest import gradient is Imports Partner Top 5% share, with log-population coefficient -0.0093 and country-clustered SE 0.0068.
  • The robustness tables separate inference, Lag-IV GMM, size-measure, annual-slope, and primary-export-share controls. The GMM check is diagnostic and should not be read as proof that endogeneity is solved.

Main specification

concentrationi,t = β log(population)i,t + γ log(GDP per capita)i,t + year FE + εi,t

Unit: country-year-flow. Inference: main table clusters by reporter country. Interpretation: negative β means larger countries have lower concentration in that outcome within the same year.

Lag-IV GMM Robustness

This check treats both log population and log GDP per capita as potentially endogenous, then instruments each with its own exact two- and three-year lags. It asks whether the country-size gradient survives an IV-GMM correction, but it does not prove that the lag instruments satisfy exclusion.

Lag-instrument specification

concentrationi,t = β log(population)i,t + γ log(GDP per capita)i,t + year FE + εi,t

Endogenous regressors: log population and log GDP per capita. Excluded instruments: each variable's exact t-2 and t-3 lags. Inference: reporter-country clustered covariance.

GMM follows Hansen (1982). The lag-instrument caution follows Roodman (2009), and the trade-diversification/development precedent is Cadot, Carrere, and Strauss-Kahn (2011).

Primary Export-Share Control

These add a country-year control for the share of exports in HS6 primary products. The strict control covers raw or very lightly processed primary goods; the broad control adds first-stage commodity processing such as refined fuels, pulp, and basic metals.

Added-control specification

concentrationi,t = β log(population)i,t + γ log(GDP per capita)i,t + θ primary export sharei,t + year FE + εi,t

Purpose: check whether the country-size gradient is mostly a commodity-exporter composition effect. HS6 999999 is excluded from both numerator and denominator before primary shares are computed.
